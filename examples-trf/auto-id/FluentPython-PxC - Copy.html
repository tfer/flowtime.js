The Python Data Model

A single chapter about the Python data model explaining how the special methods (e.g., repr ) are the key to the consistent behavior of objects of all types— in a language that is admired for its consistency. Understanding various facets of the data model is the subject of most of the rest of the book, but Chapter 1 provides a high-level overview.
